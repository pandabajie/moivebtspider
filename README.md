# 简介
本程序为电影BT种子采集器，仅供学习参考之用。
采集源目前只有http://www.btba.com.cn，其他站点的资源可以自行定制。

使用时，请在地址栏输入类似http://www.btba.com.cn/type/%E5%8A%A8%E4%BD%9C.html
就可以点击开始采集按钮了。
